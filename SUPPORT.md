# Support

Need help navigating or using Gaguinhos repositories, or our CI pipeline?

## 1. Documentation is Your Friend
The primary source of truth is the `README.md` or `.github` documentation in the respective repositories. We strive to provide comprehensive explanations there. 

## 2. Opening an Issue
If you encountered a bug or need a feature to be supported in our Universal CI, please go to the [Issues](../../issues) page of the relevant repository and select the appropriate template (Bug or Feature). 

- Describe your problem as clearly as possible.
- Provide a reproducible scenario or error logs.

## 3. Contact the Community
If you're stuck, you can reach out via our internal channels (e.g., Discord or Slack). Do not ping developers directly for help; use the public/support channels so everyone can learn together.
