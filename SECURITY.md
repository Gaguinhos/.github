# Security Policy

## Supported Versions

The Gaguinhos organization supports the latest major versions of our active projects. Security updates are applied continuously on the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them to our security team via GitHub's private vulnerability reporting feature (if enabled on the repository) or directly email us at `security@gaguinhos.local` (please update with actual email). 

We value the work of security researchers in improving the security of Gaguinhos projects. We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress fixing the vulnerability.

If a fix is implemented, we'll request a CVE and publicly acknowledge your responsible disclosure upon coordinating the release of the patched version.
